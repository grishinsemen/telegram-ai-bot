# Папка для аудиозаписей

Эта папка содержит все аудиозаписи, созданные через MiniMax Text2Speech API.

## Формат файлов

- **audio_YYYYMMDD_HHMMSS.mp3** - аудиофайлы в формате MP3
- **audio_YYYYMMDD_HHMMSS.json** - метаданные и полный ответ API для каждого аудиофайла

## Использование

Все новые аудиозаписи автоматически сохраняются в эту папку при использовании скрипта `text_to_speech.py`.

## Пример

```bash
python text_to_speech.py "Ваш текст здесь"
```

Файлы будут сохранены как:
- `audio/audio_20251122_143025.mp3`
- `audio/audio_20251122_143025.json`
