# Как использовать клонированный голос из вашего аккаунта MiniMax

## Способ 1: Найти Voice ID в веб-интерфейсе MiniMax

1. Зайдите на [platform.minimax.io](https://platform.minimax.io/)
2. Войдите в свой аккаунт
3. Перейдите в раздел **Voice Cloning** или **Мои голоса**
4. Найдите ваш сохраненный клонированный голос
5. Скопируйте **Voice ID** (обычно это строка вида `your-voice-id-123`)

## Способ 2: Использовать скрипт для обновления конфигурации

После того, как вы узнали Voice ID:

```bash
python update_voice_id.py YOUR_VOICE_ID
```

Например:
```bash
python update_voice_id.py my-cloned-voice-123
```

## Способ 3: Редактировать конфигурацию вручную

1. Откройте файл `tts_config.json`
2. Найдите строку:
   ```json
   "voice_id": "male-qn-qingse"
   ```
3. Замените на ваш Voice ID:
   ```json
   "voice_id": "ваш-клонированный-voice-id"
   ```
4. Сохраните файл

## Способ 4: Использовать интерактивный редактор

```bash
python tts_settings.py --edit
```

Введите ваш Voice ID когда будет запрошен.

## Проверка

После обновления Voice ID проверьте настройки:

```bash
python tts_settings.py
```

Убедитесь, что в разделе "Voice ID" указан ваш клонированный голос.

## Использование

После настройки просто используйте скрипт как обычно:

```bash
python text_to_speech.py "Ваш текст здесь"
```

Аудио будет создано с использованием вашего клонированного голоса!

## Важно

- Убедитесь, что ваш клонированный голос активен (использовался в течение последних 7 дней)
- Voice ID чувствителен к регистру
- Если голос не работает, проверьте, что он не истек
