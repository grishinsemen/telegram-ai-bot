# Где найти Voice ID клонированного голоса в MiniMax

## В веб-интерфейсе MiniMax

### Вариант 1: Через раздел Voice Cloning
1. Зайдите на https://platform.minimax.io/
2. Войдите в аккаунт
3. В меню найдите **Voice Cloning** или **Клонирование голоса**
4. В списке ваших клонированных голосов найдите нужный
5. Voice ID обычно отображается рядом с названием голоса

### Вариант 2: Через API (если доступно)
Если у вас есть доступ к API документации, используйте Get Voice API для получения списка.

### Вариант 3: Из истории использования
1. Перейдите в раздел **История** или **API Calls**
2. Найдите запросы, где использовался ваш клонированный голос
3. Voice ID будет в параметрах запроса

## Формат Voice ID

Клонированные голоса обычно имеют формат:
- `your-custom-voice-id`
- `clone-voice-123`
- Или любой другой ID, который вы указали при клонировании

## Быстрая проверка

После того, как вы нашли Voice ID, проверьте его:

1. Обновите конфигурацию:
   ```bash
   python update_voice_id.py YOUR_VOICE_ID
   ```

2. Сделайте тестовый запрос:
   ```bash
   python text_to_speech.py "Тест клонированного голоса"
   ```

3. Если все работает - голос найден правильно!
