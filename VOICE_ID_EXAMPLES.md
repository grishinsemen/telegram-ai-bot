## Как использовать

1. Откройте файл `tts_config.json`
2. Найдите поле `"voice_id"` в разделе `"voice_setting"`
3. Замените значение на нужный Voice ID из списка выше
4. Сохраните файл

Или используйте скрипт для редактирования:
```bash
python tts_settings.py --edit
```
